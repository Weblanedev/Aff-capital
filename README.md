# African Founders Fund (AFF)

A single-page marketing site for African Founders Fund, built with Next.js and Tailwind CSS. Matches the original design and color palette with modern animations.

## Colors

- **Gold** `#FDCB00` – logo, buttons, accents
- **Navy** `#202D4E` – logo text
- **Mustard** `#C9A227` – footer background

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Structure

- **Header** – AFF logo, Contact Us button (no link)
- **Hero** – Headline, tagline, description, Get Started button (no link), hero image
- **About Us** – INVESTMENT visual, mission copy
- **Our Core Services** – 6 service cards (Renewable Energy, Banking, Real Estate, Venture Capital, Healthcare, Payments)
- **Footer** – Brand, mission, Email Us button (mailto:dickson@aff.capital), Address (Lekki, Lagos), Help links (no links)

All buttons and footer links are placeholders and do not navigate anywhere.
